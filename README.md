# Scalable, Context-Sensitive Psychiatric Assessment With Large Language Models and Brief Audio Diaries

This repository contains the analysis notebooks and manuscript materials for the paper "Scalable, context-sensitive psychiatric assessment with large language models and brief audio diaries."

## Repository Contents

- `llm_prediction_per_item_per_subject.ipynb`: Estimates item-level DPDS scores using all diary text concatenated within subject.
- `llm_prediction_per_item_per_subject_per_day.ipynb`: Estimates item-level DPDS scores separately for each diary/transcript.
- `llm_prediction_per_domain_per_subject.ipynb`: Estimates broad psychopathology domain scores using all diary text concatenated within subject.
- `llm_prediction_per_domain_per_subject_per_day.ipynb`: Estimates broad psychopathology domain scores separately for each diary/transcript.
- `llm_prediction_reasoning_per_domain.ipynb`: Selects exemplar high/low subjects and generates text summaries with model-based reasoning.
- `Results/`: Zipped CSV files containing LLM ratings generated by the notebooks.

## Data Availability

The public repository does not include the study data.

The notebooks expect restricted local inputs such as:

- `Data/AAPECS.pickle`
- `Data/domains_to_questions.csv`
- `Data/domain_scores_per_subject_true_and_llm.csv` for the reasoning notebook

Study data access should be requested from the authors.

## Environment

This is a notebook-first project rather than a packaged Python library.

Recommended setup:

- Python 3.9
- Jupyter Notebook or JupyterLab
- Core Python packages used in the notebooks:
  - `openai`
  - `tiktoken`
  - `requests`
  - `numpy`
  - `pandas`
  - `jsonschema`
  - `tenacity`
  - `tqdm`
